## Google Maps

The courier tracking map uses `google_maps_flutter` and live location updates.
Set `GOOGLE_MAPS_API_KEY` before running the app, and enable the Google Maps SDK
for Android/iOS in Google Cloud. For iOS, expose the same value as an Xcode
build setting named `GOOGLE_MAPS_API_KEY`.

PowerShell example:

```powershell
$env:GOOGLE_MAPS_API_KEY="your-google-maps-key"
flutter run
```
